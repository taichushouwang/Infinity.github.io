用与方括号外面特殊字符的摘要

字符 意义 例子
\ 转义字符
^ 在字符串开始匹配 ‘^bob’ 匹配开始为bob的字符串
$ 在字符串末尾匹配 ‘com$’ 匹配末尾为com的字符串
. 匹配除换行符(\n)之外的字符 ‘.at’ 匹配cat, sat, mat等，只匹配单个字符
选择分支的开始(读为或)
( 子模式的开始
) 子模式的结束
* 重复0次或更多次
+ 重复1次或更多次
{ 最小/大量记号的开始
} 最小/大量记号的结束
? 标记一个子模式为可选的
